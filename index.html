การอุ้ม

ใช้งานไอเท็มอุ้มเพื่ออุ้มผู้เล่นใกล้เคียงหรือใช้มือเปล่าและคลิกขวาหรือกดค้างที่ผู้เล่นคนนั้น

หากย่อและใช้งานไอเท็มจะเปิดเมนูการตั้งค่าการอุ้ม

ล้มและชุบชีวิต

เมื่อผู้เล่นเลือดเหลือตามที่กำหนดจะส่งผลให้ผู้เล่นคนนั้นล้มและไม่สามารถขยับตัวได้

สามารถใช้ไอเท็มชุดปฐมพยาบาลหรือให้ผู้เล่นอื่นใช้ให้เพื่อชุบและเพิ่มเลือด

ปรับขนาดตัว

สามารถปรับขนาดตัวได้ตั้งแต่ 0 - 500 ( ทศนิยม 1 หลัก )

สามารถใช้งานไอเท็มขนาดตัวเพื่อเปิด Ui สำหรับปรับขนาดตัวหรือผ่านคำสั่ง event

ตัวอย่างคำสั่ง event entity @s "scale:170"

การตั้งค่า

สามารถตั้งค่าฟีเจอร์และรายละเอียดของฟีเจอร์ต่างๆ ได้ผ่านหน้า Ui

สามารถตั้งค่าได้เกือบทุกอย่าง

เช่นการอุ้ม การชุบชีวิต การซ่อนและแสดงชื่อ จำกัดขนาดตัว

ถอดมุมมอง

กดใช้งานไอเท็มถอดมุมองเพื่อเปิด Ui ถอดมุมมอง

เมื่อถอดมุมมอง สามารถขยับมุมมองได้ผ่านการเดิน กระโดดและย่อ

ชื่อลอย

กดใช้งานไอเท็มชื่อลอยเพื่อเปิด Ui สร้างชื่อลอย

ย่อและใช้งานไอเท็มเพื่อเปิด Ui ลบชื่อลอยที่อยู่ใกล้ที่สุดในระยะ 3 บล็อก

ตั้งค่าชื่อ

กดใช้งานไอเท็มตั้งค่าชื่อเพื่อเปิด Ui ตั้งค่าชื่อ

ชื่อสามารถตั้งเป็นอะไรก็ได้ สามาระใส่สีหรือรูปแบบพิเศษได้ตามปกติ

รูปแบบข้อความ จะมีข้อความที่จะถูกแทนที่ โดยในการตั้งค่าจำเป็นต้องมี ข้อความ เสมอ

{name} = ชื่อ

{message} = ข้อความ

1.0.1 - Stable Release

เวอร์ชั่นเสถียรและปรับปรุง

เผยแพร่ครั้งแรกวันที่ 12 พฤษภาคม 2025

ผ่านดิสคอร์ด Star Store สำรวจ

เปิดให้ดาวน์โหลดผ่านทางดิสคอร์ด Star Store สำรวจ

โค้ดทั้งหมดถูก Refactor มาใช้งาน Typescript แทนที่ Javascript

ปัจจุบันไม่สามารถดาวน์โหลดได้เนื่องจากการอัพเดตเวอร์ชั่นที่ใหม่กว่า

รายละเอียดเวอร์ชั่น

1. แก้ไขระบบล้มเกือบยกแผง

- เพิ่มข้อความบอกเวลาที่เหลืออยู่

- ล้มเมื่อเลือดเหลือตามที่กำหนด

- เมื่อชุบไม่ทันเวลาผู้เล่นคนนั้นก็จะเสียชีวิต

2. แก้ระบบอุ้มให้เสถียรมากขึ้น

- ลดอาการค้างเมื่ออุ้มอีกฝ่ายหรือถูกอุ้ม

- การอุ้มสามารถใช้มือเปล่าอุ้มได้ทันที

3. ถอดมุมกล้อง

- เพิ่มหน้า Ui ให้ใช้งานง่ายมากขึ้น

- การควบคุมใช้เดิมตามปกติแทนแค่กระโดดและย่อ

4. ขนาดตัว

- ลดลิมิตการปรับขนาดตัวเป็น 0 - 500 (จาก 0 - 1000)

เพื่อลดความหน่วงในเซิร์ฟเวอร์ขนาดกลางขึ้นไป

- สามารถใช้เว็บไซต์เพื่อลดหรือเพิ่มลิมิตได้

5. เพิ่มการการตั้งค่า

- จำนวนเลือดที่จะเหลือก่อนจะล้ม

- ระยะเวลาขณะล้มก่อนจะเสียชีวิต

- แก้ไขข้อความที่แสดงขณะล้ม (เปิด - ปิดได้)

- ปิด - เปิดไม่ให้ผู้เล่นอุ้มกันได้

- จำนวนเลือดที่จะได้รับหลังชุบชีวิต

6. Utils

- แก้ไขกล่องข้อความขนาดใหญ่กว่าที่ควรเป็นในทุก Ui

- ไอเท็มจะไม่ถูกลดในโหมดสร้างสรรค์เมื่อใช้ไอเท็มปรับขนาดตัวหรือชุดปฐมพยาบาล

1.5.0 - Performace Release Caching and Cacher

เวอร์ชั่นเพิ่มประสิทธิภาพและความเสถียร

เผยแพร่ครั้งแรกวันที่ 12 ธันวาคม 2025

ผ่านดิสคอร์ด Star Store สำรวจ

เปิดให้ดาวน์โหลดผ่านทางดิสคอร์ด Star Store สำรวจ

โค้ดเกือบทั้งหมดถูก Refactor อีกครั้งเพื่อประสิทธิภาพที่ดีขึ้น

ปัจจุบันสามารถดาวน์โหลดได้และเป็นเวอร์ชั่นปัจจุบัน

รายละเอียดเวอร์ชั่น

1. ขนาดตัว

- สามารถปรับขนาดตัวผ่านคำสั่งในเกม ( คำอธิบายใต้ล่าง )

2. ล้มและชุบ

- สามารถปิดความเสียหายสำหรับผู้ที่ล้ม

3. อุ้ม

- เปลี่ยนมาใช้รูปแบบ Cached Statement

- ป้องกันการอุ้มซ้อนกันซึ่งส่งผลให้เกมค้างและ Crash

- การย่อขณะใช้ไอเท็มอุ้มจะเปิดเมนูตั้งค่า

4. เมนูตั้งค่าการอุ้ม

- เปิด/ปิด การอุ้ม ( จะไม่สามารถอุ้มหรือถูกอุ้มได้, สามารถปิดได้ในตั้งค่า )

- เปิด/ปิด การเลิกอุ้มเมื่อย่อ ( ต้องใช้ไอเท็มในการเลิกอุ้มเท่านั้น )

5. การลบชื่อลอย

- เพิ่มเมนูแจ้งเตือนเมื่อไม่พบชื่อลอย

- แสดงรายละเอียดของชื่อลอยในเมนูการลบ

6. กล้อง

- เปลี่ยนมาใช้รูปแบบ Cache

- เมื่อปิดใช้งานจะมีอนิเมชั่นถูกดึงกลับมาที่ส่วนหัว

7. ตั้งค่าชื่อและข้อความ

- สามารถตั้งชื่อได้ ( สามารถปิดได้ในตั้งค่า )

- สามารถเปลี่ยนรูปแบบข้อความแชทได้ ( สามารถปิดได้ในตั้งค่า )

8. เพิ่มการตั้งค่า

- สามารถทำความเสียหายต่อผู้ที่ล้ม ( เปิด - ปิด )

- สามารถแก้ไขชื่อได้ ( เปิด - ปิด )

- สามารถแก้ไขรูปแบบข้อความได้ ( เปิด - ปิด )

9. ไอเท็ม

- ไอเท็มถูกจัดหมวดหมู่รวมกัน

- เพิ่มไอเท็ม ชื่อ สำหรับเปิดเมนูตั้งค่าชื่อและข้อความ

- แก้ไขไอค่อนของไอเท็มอุ้ม

- แก้ไขไอค่อนของไอเท็มถอดมุมมอง

- แก้ไขไอค่อนของไอเท็มชุดปฐมพยาบาล

- แก้ไขไอค่อนของไอเท็มขนาดตัว

10. Utils

- Refactor และเพิ่มระบบ Cache เพื่อประสิทธิภาพที่ดีมากขึ้น

- เปลี่ยนมาใช้ Beta Api ( จำเป็นต้องเปิดตั้งค่า Beta API ภายในแมพ )
